What are we doing

App (Ios, Andriod and Web-site) used for onboard checks and attendance. The user creates a "portfolio" while in the office, and attaches to it the profiles selected from the list. Questionnaires will be answered on board. The portfolio returned is called an "attendance report". Finally, upon returning to the office, the attendance report is transferred to the office's database server for further use.

Ios and Android

The applications are designed according to the latest trends in coding, we also provided for data compression, so as not to cause discomfort, because at sea, a person can be a long time and accumulate a lot of photos that need to send on arrival in port

Web-version

For the site we have developed an adaptive design, it will look cool from a laptop, tablet and phone. For best speed and performance we have implemented it with Javascript

Admin panel

The entire ecosystem can be managed from a convenient admin panel: Create users, allow or disallow creating briefcases, view sent cases, download data in a convenient format for further work
